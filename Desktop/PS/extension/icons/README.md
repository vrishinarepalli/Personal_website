# Extension Icons

To complete the extension, add icon files:
- icon16.png (16x16)
- icon32.png (32x32)
- icon48.png (48x48)
- icon128.png (128x128)

For now, you can use any Pokemon-themed icons or create simple placeholder icons.

You can use sites like:
- https://www.flaticon.com/
- https://icon-icons.com/
- Or create your own with a Pokemon ball design
